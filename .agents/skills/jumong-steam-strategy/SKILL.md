# 주몽 Steam 전략

## 소유 영역

- `docs/production/research/steam/`
- Steam 출시 전략과 시장 조사 브리프
- 전문 에이전트별 영향 검토 요청과 의견 통합

## 조사 원칙

1. 최신 Steamworks 공식 문서와 Valve 공지를 우선한다.
2. 시장 규모와 전환율은 분석업체 추정치임을 명시하고 공식 사실과 분리한다.
3. 출처 URL, 게시일 또는 확인일, 적용 범위와 불확실성을 기록한다.
4. 트렌드를 제작 명령으로 취급하지 않고 주몽의 전체 2D 정체성과 상업 목표에 맞는지 검토한다.
5. 기본 주기는 주 1회와 각 마일스톤 시작 시점으로 한다.
6. 다국어는 한국어 원문, 영어 1차 출시를 기본 가설로 두고 위시리스트·지역별 관심·번역 비용·QA 가능성에 따라 추가 언어 우선순위를 제안한다.

## 협의 절차

1. 조사 브리프를 작성한다.
2. narrative, level-2d, character, 2d-art, core-systems, ui-systems, combat, audio 담당에게 영역별 영향을 요청한다.
3. 각 의견을 적용 가치, 비용, 위험, 보류 이유로 정리한다.
4. `$jumong-game-director`가 공통안과 이견을 통합하게 한다.
5. `$jumong-production-governor`가 전체 2D 방향과 파일 소유권을 감사하게 한다.
6. 결정이 필요한 항목만 일일 보고서에 올리고 나머지 조사·문서 동기화는 계속한다.

## 사용자 결정 경계

- 출시일, 가격, 할인율, 데모 공개 시점, Next Fest 사용 시점
- 상점 페이지 공개, 태그 정본, 마케팅 문구와 외부 게시
- 지원 언어 범위, 최종 AI 콘텐츠 정책, 유료 마케팅 또는 외부 계약

위 항목을 직접 변경하거나 게시하지 않는다. 선택지, 추천, 비용과 영향을 제시해 사용자 결정을 받는다.
