# 주몽 오디오

## 소유 영역

- `docs/audio/`
- `data/audio/`
- `assets/audio/`
- `tools/audio/`
- 오디오 재생 전용 스크립트와 버스 설정

## 작업 절차

1. 씬·컷신·전투의 큐 ID, 시작 조건, 중지 조건과 반복 정책을 정의한다.
2. 파일 경로와 런타임 데이터를 분리하고 누락 파일 처리를 명시한다.
3. `AudioStreamPlayer` 계열과 오디오 버스를 사용하며 3D 공간 음향을 제작 기준으로 도입하지 않는다.
4. 음량 기준, 페이드, 동시 재생 제한과 루프 경계를 검토한다.
5. 로그와 실제 청취가 필요한 검증 항목을 분리해 전달한다.

## 무료 음원 조달

1. 필요 큐와 대체 불가능성을 먼저 정의하고 직접 제작·구매·무료 조달을 비교한다.
2. 사이트 설명이 아니라 개별 파일의 공식 라이선스를 확인하고 URL, 제작자, 파일명, 다운로드일, 상업 이용, 수정, 출처표기, 재배포, 생성형 AI와 Content ID 조건을 `docs/audio/`의 출처 대장에 기록한다.
3. CC0이며 권리 출처가 명확한 파일만 승인 범위 안에서 자율 도입한다. CC BY, 사이트 자체 라이선스, 로그인 필요, 독점 조항, AI 생성, 음성과 Content ID 가능성은 일일 보고의 결정 게이트로 올린다.
4. CC BY-NC, CC BY-ND, personal/editorial-only, 출처 불명, 게임·영상 추출 파일은 사용하지 않는다.
5. 원본 단독 재배포를 금지하는 라이선스는 공개 Git 저장소에 커밋하지 않고 별도 승인과 패키징 계획을 요구한다.

## 협업 경계

- 컷신 사건 시점은 `$jumong-narrative`와 `$jumong-level-2d`가 정의한다.
- 옵션 화면은 `$jumong-ui-systems`가 소유한다.
- 설정 저장은 `$jumong-core-systems`가 소유한다.
- 생성형 음악·음성·효과음은 AI 콘텐츠 대장과 비침해 검토를 갱신한다.
