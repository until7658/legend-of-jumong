# 주몽 2D 아트

## 2D 캐릭터 중심 규칙

- 캐릭터, 접지 그림자, UI와 핵심 VFX는 2D 래스터 또는 Godot 2D 셰이더·파티클로 제작한다.
- 최종 화면과 캐릭터 아트 파이프라인은 2D가 기준이다. 승인된 교전 컷인의 작은 3D 배경과 3D 기반 이펙트는 2D로 합성되는 예외로 관리하며 원본, 사용 위치, 2D 폴백, 성능과 라이선스를 별도 기록한다.
- 깊이감은 기본적으로 레이어, 명암, 대기원근, 패럴랙스와 2D 후처리로 표현하고, 승인된 미니 필드는 제한 팔레트·무광 재질·고정 직교 카메라로 동일 화풍을 맞춘다.

## 소유 영역

- `assets/characters/`
- `assets/maps/` 및 2D 배경·타일·프롭 영역
- `docs/art/`
- 아트 관련 데이터 매니페스트
- `docs/assets/ai_asset_log.md`의 관련 항목

## 작업 절차

1. 승인된 비주얼 바이블과 사용 씬을 확인한다.
2. 해상도, 피벗, 방향, 프레임, 투명 배경, 임포트 필터를 명시한다.
3. 플레이스홀더와 최종 후보를 분리한다.
4. 생성형 애셋은 도구, 날짜, 목적, 수정 여부, 사용 위치, 비침해 검토를 기록한다.
5. 실제 씬 통합과 화면 가독성은 담당 에이전트 및 `$jumong-verify`와 확인한다.

## 협업 경계

- 캐릭터 씬 연결은 `$jumong-character`가 담당한다.
- 맵 배치는 `$jumong-level-2d`가 담당한다.
- UI 레이아웃은 `$jumong-ui-systems`가 담당한다.
