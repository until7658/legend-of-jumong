# 북방 자연환경 카툰 골든 샘플 검토

## 목적과 상태

캐릭터 원화 `assets/concept/cartoon_v1/`과 같은 화면에 놓일 북방 자연환경의 렌더링 언어를 승인하기 위한 스타일 샘플이다. 파일명은 요청된 `gold_v1`을 따르지만, **Terrain Set 또는 게임 채택 최종본이 아니라 사용자·PM 승인 후보**다.

- 지면 시트: `assets/tiles/golden/cartoon_northern_ground_gold_v1.png`
- 자연물 시트: `assets/objects/golden/cartoon_northern_props_gold_v1.png`
- 기준 원화: 주몽, 유화, 금와의 `cartoon_v1`
- 생성일: 2026-07-17

## 적용한 통합 스타일

- 가늘고 날카로운 어두운 선
- 밝음·중간·암부의 2.5단 셀 채색
- 큰 색면과 각진 실루엣
- 재질을 설명하는 범위로 제한한 마른 붓 질감
- 저채도 녹갈색·회갈색·회청색 환경 팔레트
- 청흑색 암부와 일부 상단 모서리에만 얇은 호박색 새벽 림라이트
- 캐릭터보다 낮은 채도·대비·디테일 빈도

## 지면 샘플 검토

풀, 다져진 흙, 얕은 물, 층상 바위와 서로의 접경을 한 화면에서 비교할 수 있다. 정사영에 가까운 카메라와 좌상단 광원을 사용한다.

적합한 점:

- 네 재질이 큰 명도·색상 면으로 즉시 구분된다.
- 물과 바위의 청회색이 캐릭터의 적갈·남청 진영색과 경쟁하지 않는다.
- 바위의 청흑 암부와 얇은 호박 가장자리가 캐릭터 원화의 광원 언어와 연결된다.
- 사진 질감과 3D 재질 광택은 억제되어 있다.

후속 보완:

- 풀잎과 흙 균열은 실제 게임 크기에서 한 단계 더 단순화한다.
- 현재 이미지는 연속된 환경 콘셉트이며 256px 셀, 무봉합 경계, Terrain peering 규칙을 제공하지 않는다.
- 1024×1024 4×4 정규화 아틀라스 제작 전 3×3 반복 검사가 필요하다.

## 자연물 샘플 검토

바람에 휜 소나무, 성긴 낙엽수, 각진 바위 무더기 2종, 갈대와 마른 풀을 독립 실루엣으로 비교한다.

적합한 점:

- 나무 수관과 바위가 큰 덩어리로 읽혀 작은 화면에서도 종류를 구분할 수 있다.
- 접지점이 하단 중앙에 모여 향후 Y 정렬 기준을 잡기 쉽다.
- 인물 원화보다 채도는 낮지만 선과 면 분할 방식은 공유한다.
- 문자, 인물, 건물, 상징과 특정 작품을 연상시키는 요소가 없다.

후속 보완:

- 생성 결과가 불투명 회백 배경이므로 실제 게임 사용 전 개별 오브젝트를 수작업 분리하거나 투명 원본으로 재제작한다.
- 가는 잔가지와 침엽수 잎은 필드 크기에서 뭉칠 수 있어 양산본에서는 20~30% 단순화한다.
- 바위 접지부의 작은 파편은 충돌 실루엣에서 제외한다.
- 자연물 시트는 규격·피벗·충돌이 없는 콘셉트 시트이며 Sprite2D 원본으로 직접 사용하지 않는다.

## 캐릭터 대비 판단

원화의 가장 강한 특징인 짙은 머리·외곽선, 주몽의 적갈색, 금와의 남청색보다 환경 명도 대비와 채도를 낮게 유지했다. 따라서 통합 화면에서는 캐릭터가 한 단계 선명하게 읽힐 가능성이 높다. 다만 실제 판정에는 다음 합성 증거가 필요하다.

1. 1280×720 제1장 궁술 훈련장 화면
2. 실제 필드 크기의 주몽 정지 스프라이트
3. 동일한 좌상단 새벽광과 그림자
4. 지면·나무·바위·표적을 함께 배치한 화면
5. 컬러 화면과 저채도/명도 확인본

## 판정

**승인 후보 / 부분 충족**

캐릭터와 공유할 환경 스타일 방향은 제시되었다. PNG 무결성, 이미지 크기와 기본 구성은 확인했다. 그러나 무봉합 타일, 투명 오브젝트, 반복성, 실제 캐릭터 합성 가독성은 아직 검증되지 않았으므로 Terrain Set 또는 최종 게임 애셋으로 승격할 수 없다.
