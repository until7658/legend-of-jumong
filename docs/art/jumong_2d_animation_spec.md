# 주몽 2D 캐릭터 애니메이션 사양 v1

## 적용 기준

- 기준 원화: `assets/concept/cartoon_v1/jumong.png`
- 직업: 바람을 읽는 궁수
- 전투 역할: 후열 정밀 공격, 표식, 원거리 엄호
- 무기: 합성궁과 비상용 소형 칼
- 기준 캔버스: 프레임당 384×384 투명 PNG
- 피벗 기준: 수평 중앙 `x=192`, 지면 접점 `y=340`
- 검수 씬: `scenes/characters/jumong_animation_preview.tscn`

## 프로토타입 제한

- 생성형 원본을 알파 연결 성분으로 분리하고 동일 캔버스에 정규화한 1차 결과다.
- 4프레임 보행은 동작 폭이 작아 최종 제작에서는 접지, 상하 운동과 팔·활의 역운동을 수작업 보정해야 한다.
- `차단 사격`, `꿰뚫는 화살`, `결의의 일시`는 전투 시스템의 타이밍과 이펙트 사양이 확정된 뒤 별도 애니메이션으로 확장한다.
- 발사 화살과 적중 이펙트는 캐릭터 스프라이트에 고정하지 않고 전투 시스템의 별도 투사체·이펙트로 분리하는 것을 원칙으로 한다.
