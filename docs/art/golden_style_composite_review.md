# 골든 스타일 통합 합성 검수 v1

## 검수 대상과 범위

- 결과 이미지: `assets/review/golden_style_composite_v1.png`
- 해상도: 1280×720
- 가정: 45~50도 하향 전체 2D 쿼터뷰, 일반 탐색 화면
- 캐릭터: `jumong_gold_idle_sheet_v1.png`의 첫 번째 대기 프레임 1개만 사용
- 캐릭터 화면상 유효 신장: 약 145 px
- 배경: `cartoon_northern_ground_gold_v1.png`를 1280×720 화면에 맞게 크롭·스케일
- 자연물 진단: `cartoon_northern_props_gold_v1.png`의 소나무 일부를 원본 배경 그대로 삽입

이 합성은 신규 생성형 콘텐츠가 아니다. 기존 에셋의 크롭, 스케일링, 접지 그림자와 진단용 표기만 사용한 정적 검수 이미지다. 실제 Godot 카메라, 조명, 충돌, Terrain 반복 배치를 검증하지 않는다.

## 중요 전제

`cartoon_northern_ground_gold_v1.png`는 1254×1254 단일 회화형 보드다. Godot Terrain Set이나 반복 가능한 타일 시트가 아니므로, 이 이미지가 자연스럽게 보이는 것만으로 맵 제작 가능성을 승인할 수 없다.

## 관찰 결과

### 캐릭터 크기와 접지

- 145 px 신장은 스타일 바이블의 일반 탐색 권장 범위 110~155 px 안에 있다.
- 머리, 적갈색 상의, 활의 방향은 1280×720 전체 화면에서도 구분된다.
- 짧은 반투명 접지 그림자를 추가하면 발 위치는 읽히지만, 원본 프레임의 발 주변 녹색 프린지가 접지선을 깨뜨린다.
- 캐릭터 자세가 정면 필드 대기라기보다 전신 원화 축소에 가까워, 내려다보는 카메라에서 상체와 발의 원근이 충분히 맞지 않는다.

### 배경 채도·대비

- 적갈색 주몽은 회갈색 흙길 위에서 대체로 읽힌다.
- 풀 지역에서는 녹색 프린지와 배경 초록 계열이 합쳐져 머리·어깨 외곽이 불안정하다.
- 배경은 중저채도라 팔레트 방향은 적합하지만, 풀잎·자갈·암석 균열의 미세 묘사가 캐릭터보다 선명한 구간이 있다.
- 캐릭터 주변 24~40 px의 시각 잡음을 낮추는 별도 플레이 공간 정리가 필요하다.

### 외곽선과 렌더링 언어

- 배경과 캐릭터 모두 붓 질감은 공유하지만 캐릭터는 축소 후 내부 세부가 뭉치고 외곽선이 약해진다.
- 녹색 프린지는 투명 PNG 가장자리 전반에 남아 있으며 실제 배경에서 명확히 보인다.
- 캐릭터는 스타일 바이블이 요구하는 스프라이트 전용 2.5단 셀 채색보다 사실적 전신 일러스트에 가깝다.

### 자연물 알파

- 자연물 시트는 `Format24bppRgb`이며 알파 채널이 없다.
- 합성 우측 상단에 불투명한 미색 배경을 그대로 노출했다. 이 상태로는 맵 오브젝트로 사용할 수 없다.
- 자동 색상 제거로 숨길 문제가 아니다. 나뭇가지·잎 가장자리의 수동 알파 정리와 게임 배경 위 프린지 검사가 필요하다.

## 판정

**최종 판정: 실패 — 골든 스타일 승인 불가**

### P0 — 통합 전에 반드시 수정

1. 주몽 대기 프레임의 녹색 프린지를 제거하고 투명 가장자리를 수동 검수한다.
2. 자연물 시트에 올바른 알파를 제작하고 밝은색·어두운색·녹색 배경에서 가장자리를 확인한다.
3. 단일 회화 보드를 Terrain으로 오인하지 않도록, 반복 가능한 풀·흙·물가 전환 셀과 4×4 반복 미리보기를 별도로 제작한다.

### P1 — 수직 슬라이스 전에 수정

1. 주몽을 원화 축소가 아닌 145 px 표시용 필드 스프라이트로 단순화한다.
2. 머리·어깨·활 외곽선을 현재보다 강하게 하고 내부 복식 세부를 줄인다.
3. 플레이 경로 주변의 풀·자갈 밀도를 낮춰 캐릭터와 최소 20% 국소 명도 차를 확보한다.
4. 실제 Godot 직교 카메라에서 접지점, Y 정렬, 그림자와 100%·75%·50% 표시를 캡처한다.

### P2 — 양산 단계에서 개선

1. 물·바위·풀의 디테일 밀도 차이를 지역 규칙으로 문서화한다.
2. 새벽·낮·흐림 조명에서 적갈색 진영색의 가독성을 비교한다.

## 다음 승인 조건

- 프린지가 제거된 주몽 대기 프레임 1개
- 알파가 정리된 나무 1종과 바위 1종
- 반복 가능한 소규모 Terrain 골든 세트와 4×4 반복 미리보기
- 위 에셋을 실제 Godot 직교 카메라에 넣은 1280×720 무보정 캡처
- 오류를 감추는 후처리 없이 밝은색·어두운색·녹색 배경 비교 화면 제출

사용자와 PM이 위 화면을 확인하기 전에는 캐릭터·자연물·Terrain 대량 제작으로 확장하지 않는다.
