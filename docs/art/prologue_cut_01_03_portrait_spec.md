# 오프닝 1~3컷 전용 일러스트 사양

## 연출 원칙

- 실제 맵과 필드 캐릭터 행동을 기본 화면으로 사용한다.
- 내레이션에는 초상화를 표시하지 않는다.
- 인물 대사가 발생할 때만 화자 전용 흉상 일러스트를 대화 UI에 표시한다.
- 최종 스타일은 카툰형 02의 거친 외곽선과 2~3단 셀 명암에 맞춘다.

## 추가 제작 우선순위

- P0 `fisherman/alert_discovery`: 물가의 형체를 발견한 경계와 놀람.
- P0 `fisherman/breath_check_concern`: 호흡을 확인하는 긴장, 과장된 공포 금지.
- P0 `yuhwa/unconscious_wet`: 젖은 머리와 의식 없는 구조 자세.
- P1 컷 2 그물 회수 반신 행동 삽화.
- P1 컷 3 어부와 유화 구조 장면 키 아트.

현재 `fisherman/concerned.png`는 임시 사용 가능하다. `yuhwa/calm.png`는 컷 3의
젖고 의식 없는 상태와 맞지 않으므로 필드 캐릭터 플레이스홀더만 사용하고 대화 초상화로
표시하지 않는다.
