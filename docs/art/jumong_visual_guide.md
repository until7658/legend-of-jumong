# 주몽신화전기 초기 비주얼 가이드

## 방향

- 역사적 사실을 단정하기보다 확인 가능한 북방계 생활 요소와 게임 각색을 구분한다.
- 특정 영화, 드라마, 게임의 의상·배우·화면 구성을 복제하지 않는다.
- 실용적인 복식과 사용감 있는 재료를 우선하고 장식은 사회적 지위를 설명할 만큼만 사용한다.
- 최종 인물 방향은 사실적인 렌더보다 고전 한국 판타지 RPG를 연상시키는 카툰 일러스트로 잡는다. 선명한 윤곽선, 셀 채색, 선택적인 붓 질감, 약간 길게 잡은 인체 비례와 절제된 표정을 사용한다.
- 특정 게임이나 작가의 고유 화풍·얼굴·의상을 그대로 복제하지 않고, 우리 작품의 복식·색상·실루엣 체계로 독자성을 확보한다.
- 프로젝트 외부 참고 이미지 `C:/Project/참고이미지/화풍1.png`, `화풍2.png`에서 가늘고 날카로운 선, 각진 얼굴과 눈매, 셀 채색과 부드러운 그라데이션의 혼합, 강한 암부, 크림·호박색 림라이트, 바람에 흐르는 머리카락과 천의 역동성을 스타일 기준으로 추출한다.
- 참고 이미지에 포함된 캐릭터, 무기, 문신, 문양, 포즈, 구도, 문자와 로고는 사용하지 않는다.
- 세력 구분색은 동부여의 청색 계열과 주몽의 적색 계열로 나눈다. 같은 계열 안에서도 신분과 직업에 따라 명도·채도를 달리한다.
- 중국식 장포로 읽히는 바닥까지 내려오는 넓은 겉옷과 과도한 교차 여밈을 피한다. 짧은 저고리형 상의, 활동적인 바지, 허리띠와 대조색 선 장식 등 한국 고대 북방계 복식에서 착안한 실루엣을 우선한다.

## 세력 색상

- 동부여 왕실: 짙은 남색·청흑색, 제한적인 청동색과 회백색 장식
- 동부여 관리: 먹청색·회청색, 신분을 나타내는 좁은 선 장식
- 동부여 백성: 빛바랜 쪽빛·회청색, 흙과 물에 바랜 낮은 채도
- 주몽: 초반에는 탁한 적갈색·벽돌색, 성장하며 선명한 적색과 적동색을 점진적으로 추가
- 유화: 동부여 소속이 아니므로 옅은 회백색을 중심으로 하되, 동부여에서 지급받은 옷에는 연한 청색을 제한적으로 사용

## 유화 콘셉트

- 장면: 구조 후 궁의 독립 처소에서 의식을 되찾은 직후
- 표정: 혼란과 경계, 육체적 피로
- 의상: 질 좋은 옅은 회청색 겹옷이 젖고 일부 훼손된 상태
- 장식: 작은 청동 귀걸이 또는 옷고름 장식 하나
- 조명: 창호를 통과한 자연스러운 아침 햇빛
- 금지: 광륜, 마법 입자, 왕관, 현대 화장, 과도한 노출, 화려한 판타지 갑옷

## 17세 주몽 콘셉트

- 시기: 제1장 궁술 시험부터 동부여 탈출 직전, 17세
- 표정: 경계심, 침착함, 결의
- 의상: 수선 흔적이 있는 탁한 적갈색의 짧은 저고리형 상의와 회갈색 바지. 동부여의 청색과 즉시 구분되어야 한다.
- 장비: 가죽 팔보호대, 비대칭 활집, 실용적인 합성궁과 소형 칼
- 체형: 마르고 민첩한 궁수 체형
- 금지: 왕관, 삼족오·태양 문장, 금색 판금갑옷, 과장된 근육, 현대적 헤어스타일, 타 작품을 연상시키는 디자인

## 전체 인물 제작 기준

- 전체 인물의 얼굴, 체형, 성격, 복식, 장비와 스프라이트 동작은 `docs/characters/full_character_design_bible.md`를 기준으로 한다.
- 대화 초상화는 기본·기쁨·경계·분노·슬픔·결의 6종을 원칙으로 하되, 단역은 장면에 필요한 표정만 제작한다.
- 필드 및 전투 스프라이트는 작은 화면에서 머리 모양, 어깨선, 무기와 색상만으로 구분되어야 한다.

## 활용 제한

생성 이미지는 캐릭터 콘셉트 탐색용 초안이다. 최종 게임 원화·스프라이트로 채택하기 전에 인체, 복식, 무기 구조와 상업 사용 적합성을 별도로 검토한다.

현재 사용자 확인 대상은 `assets/concept/cartoon_v1/`의 참고 화풍 반영본 6종이다. 기존 `assets/concept/prologue/`와 `jumong_fullbody_concept.png`는 비교 및 변경 이력용 사실적 초안으로만 보존한다.
