# 주몽 2D 골든 샘플 검토 v1

## 목적과 판정 범위

이 문서는 17세 주몽의 대기·정면 걷기·기본 사격 3종을 동일한 2D 제작 언어로 시험한 골든 방향성 검토다. 대량 애니메이션 제작이나 Godot 통합 승인이 아니다.

- 정본 인물·스타일: `assets/concept/cartoon_v1/jumong.png`
- 보조 비교: `assets/concept/main_cast_v1/jumong.png`
- 스타일: 거친 선화, 2.5단 셀 채색, 제한적인 마른 붓 질감
- 규격: 프레임당 384×384, 4열 시트 1536×384, 피벗 `(192, 340)`
- 공통 특징: 적갈색 상의, 청흑색 하의와 암부, 높은 검은 묶음머리, 합성궁과 화살통
- 상태: `gold`라는 파일명은 요청된 산출물명일 뿐, 사용자·PM 승인 전 제작 상태는 **RC/검토용**이다.

## 공통 검토

### 충족

- 세 시트 모두 17세의 젊고 마른 궁수 실루엣, 높은 묶음머리와 적갈색 외투가 읽힌다.
- 기존 프로토타입보다 내부 복식 정보가 줄어 384px 셀에서 얼굴·활·세력색의 우선순위가 명확하다.
- 각 프레임을 384×384 셀에 정규화했고 접지 기준을 `y=340`에 맞췄다.
- 사진식 사실주의 대신 선화와 큰 색면 중심의 카툰 렌더링 방향을 유지한다.

### 미충족

- 크로마 제거 뒤 머리카락·활·옷자락 가장자리에 녹색 프린지가 남는다.
- 생성형 결과에서 얼굴과 머리카락 가닥, 옷자락 파손 모양, 활 세부가 프레임마다 조금씩 바뀐다.
- 캐릭터 세부가 목표한 40~60%보다 덜 단순화되어 실제 축소 화면에서는 여전히 밀도가 높다.
- 활·시위·화살을 몸체와 한 번에 생성해 무기 형태를 기계적으로 고정할 수 없다.

## 동작별 판정

### 대기

- 장점: 얼굴, 몸 비례, 접지와 합성궁의 전체 실루엣이 네 프레임에서 가장 안정적이다.
- 결함: 호흡 동작이 거의 읽히지 않고, 활과 옷자락이 미세하게 다른 형태로 다시 그려졌다.
- 판정: **부분 충족**. 정지 자세와 축소 실루엣 기준으로만 사용할 수 있다.

### 정면 걷기

- 장점: 좌우 다리 교대와 전진 자세가 읽히고, 정면 얼굴과 적갈색 상체가 유지된다.
- 결함: 팔의 역운동이 약하고 네 번째 프레임의 다리 간격·활 위치·하의 주름이 다른 프레임과 어긋난다. 1·3 및 2·4가 자연스러운 대응쌍이 아니다.
- 판정: **부분 충족**. 보행 방향·실루엣 참고용이며 반복 재생용 프레임으로 사용할 수 없다.

### 기본 사격

- 장점: 준비 → 거상·노킹 → 만작 → 릴리스의 서사적 순서는 명확하다.
- 결함: 활 길이와 곡률, 그립 위치, 시위 삼각형, 화살 위치가 프레임마다 달라진다. 프레임 경계를 넘는 활·화살 때문에 시트 분리 과정에서 이웃 프레임 잔상이 포함되며, 릴리스 자세도 동일한 골반·발 고정에서 나오지 않는다.
- 판정: **실패**. 타이밍 보드로만 참고하며 게임 애니메이션 및 합성 화면에 사용하지 않는다.

## 재작업 기준

1. 승인된 단일 정지 베이스를 몸통·머리·앞팔·뒤팔·다리·외투 자락으로 분리한다.
2. 활·시위·화살·화살통은 캐릭터와 분리된 투명 레이어로 수작업 제작한다.
3. 활은 하나의 벡터 또는 고정 래스터 원본을 변형해 모든 프레임에서 전체 길이와 팁 모양을 보존한다.
4. 걷기는 접촉·하강·통과·상승 포즈를 수작업 키로 잡고 발 접점 ±2px, 머리 높이 ±3px을 수치 검사한다.
5. 얼굴은 한 베이스에서 눈·턱·머리 윤곽을 고정하고 표정 변화만 제한한다.
6. 외투 찢김과 끈 장식은 큰 덩어리 3~5개로 단순화하고 프레임마다 새로 생성하지 않는다.
7. 투명 원본에서 제작하거나 수작업 마스크를 정리해 녹색 프린지를 제거한다.
8. 25%, 50%, 100% 크기와 골든 맵 합성 화면에서 얼굴·활·접지·세력색을 다시 확인한다.

## 다음 역할 인계

- PM·아트 디렉터: 대기 자세의 실루엣과 전체 색 비율만 골든 후보로 검토한다.
- 2D 애니메이터: 생성 시트를 직접 채택하지 말고 위 재작업 기준에 따라 분리 레이어 기반으로 다시 제작한다.
- 맵·통합 담당: 이 파일들은 정식 합성 샘플에 넣지 않는다. 필요하면 대기 1프레임만 임시 비교용으로 사용하고 `WIP`를 표시한다.
- 검증 담당: 알파 프린지, 셀 규격, 접지선과 활 길이 편차를 화면 증거로 재확인한다.

## 최종 자체 판정

**부분 충족.** 스타일 방향과 축소 실루엣 검토에는 의미가 있지만 프레임 일관성과 투명 가장자리 품질이 골든 승인 기준에 미달한다. 세 파일 모두 최종 게임 사용 불가이며, 수작업 재구성과 검증이 필요하다.
