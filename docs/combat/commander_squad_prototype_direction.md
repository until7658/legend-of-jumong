# 지휘관·분대 전술 전투 프로토타입 방향

## 결정

M4부터 전투의 기본 조작 단위는 개인 영웅이 아니라 **지휘관 1명과 병사 6~12명으로 구성된 분대**다. M4 표시 기준은 지휘관 1명과 병사 8명이다. 화면은 45~50도 하향 쿼터뷰이며, 논리 좌표는 사각 격자를 사용하고 화면에는 다이아몬드 형태로 투영한다.

랑그릿사 계열의 지휘관·용병 구조는 장르 참고점으로만 삼는다. 기존 작품의 UI, 수치, 클래스명, 맵, 대형, 연출을 복제하지 않는다. 주몽신화전기는 고대 부여·고구려의 `군령`, `대열`, `기세`를 중심으로 독자 규칙을 만든다.

## M4 최소 플레이 루프

1. 플레이어가 `주몽 훈련분대`를 선택한다.
2. 이동 가능 칸과 공격 가능 칸을 구분해 표시한다.
3. 분대를 이동한다. 지휘관을 중심으로 병사들이 쿼터뷰 대형을 유지한다.
4. 기본 공격 또는 방어 명령을 선택한다.
5. 모든 아군 분대가 행동하면 적 분대가 결정론적 AI로 행동한다.
6. 적 지휘관의 기세를 0으로 만들거나 병사 전원을 무력화하면 승리한다.

훈련전에서는 사망 대신 `제압`과 `후퇴`로 표현한다.

## 프로토타입 분대

| 진영 | 지휘관 | 병사 | 역할 |
|---|---|---|---|
| 아군 | 주몽 | 수련 궁병 8 | 거리 2~3칸에서 창병 대열을 흔든다. 인접 공격에는 불리하다. |
| 적군 | 훈련대장 | 수련 창병 8 | 인접 칸에서 궁병을 압박하고 방어 명령으로 피해를 줄인다. |

M4 상성은 읽기 쉬운 위치 규칙 하나로 제한한다.

- 궁병이 2칸 이상 거리에서 창병을 공격: 피해 보정 `+2`
- 창병이 인접한 궁병을 공격: 피해 보정 `+2`
- 방어 명령 중인 분대: 다음 피격 피해 `-1`

## 상태와 수치

분대는 다음 최소 데이터를 가진다.

- `commander_id`, `troop_type`, `side`
- `grid_position`, `move_range`, `attack_min`, `attack_max`
- `commander_morale`: 지휘관의 전투 지속력
- `soldier_count`, `soldier_morale`: 병사 수와 대열의 기세
- `acted`, `defending`, `routed`

피해는 병사 기세와 병사 수를 먼저 줄이며, 병사가 모두 제압된 뒤 남은 피해가 지휘관 기세에 적용된다. 이 구조는 지휘관과 분대가 함께 싸운다는 감각을 만들되 M4 구현 복잡도는 낮춘다.

## 전투 상태머신

`SETUP -> PLAYER_SELECT -> PLAYER_MOVE -> PLAYER_COMMAND -> RESOLVE -> ENEMY_TURN -> ROUND_END -> VICTORY/DEFEAT`

선택 취소는 한 단계 전으로 돌아가며, 결과 계산은 재현 가능한 고정 수치를 사용한다. 난수·치명타·장비·성장은 M4에서 제외한다.

## 화면 기준

- 지휘관은 약 3등신 SD 도트 캐릭터로 분대 중앙 또는 선두에 둔다.
- 전술 맵에서는 지휘관과 병사 8명을 2열 대형으로 축약 표시한다. 향후 병사 수는 6~12명 범위에서 병종·지휘력에 따라 달라진다.
- 선택 분대는 청록 외곽선, 이동 칸은 청색, 공격 칸은 적갈색으로 구분한다.
- 분대 정보에는 지휘관 이름, 병종, 병사 수, 기세만 표시한다.
- 마우스와 키보드, 컨트롤러 방향키·확인·취소·턴 종료를 같은 행동으로 연결한다.

## M4 검증 질문

1. 플레이어가 10초 안에 지휘관과 병사를 하나의 분대로 인식하는가?
2. 거리 상성을 UI 설명 없이 한 번의 교전으로 이해할 수 있는가?
3. 분대 병사 수가 줄 때 전투력이 약해졌다고 느끼는가?
4. 개별 병사 조작보다 분대 단위 조작이 주몽의 지휘관 판타지를 강화하는가?

## 이후 확장 후보

- 전투 전 병종 고용·교체와 2종 혼성 대열
- 지휘 반경 밖 병사의 기세 저하
- 기병·보병·궁병·창병의 순환 상성
- 숲·강·언덕·진흙의 이동 및 방어 효과
- 오이·마리·협보의 지휘 특성과 합동 군령
- 병사 생존률에 따른 숙련도와 다음 전투 보급 비용

## 연출 레이어 구분

- **전술 맵:** 지휘관과 분대가 한 조작 단위로 이동한다. 다수 병사를 축약 대형으로 보여 주되 선택과 격자 가독성을 우선한다.
- **교전 컷인:** 공격 확정 시 별도 짧은 2D 무대에서 지휘관과 병사 다수가 양쪽 대열로 충돌한다. 피해에 따라 병사가 이탈하고 지휘관 기술이 강조된다.
- **캐릭터 컷신:** 궁술대회, 대화, 결투, 주요 서사 장면은 분대전과 다른 캐릭터 중심 연출을 사용한다. 궁술대회에서는 주몽 개인의 자세·호흡·화살 궤적과 관중 반응을 보여 주며 분대 대형을 사용하지 않는다.
