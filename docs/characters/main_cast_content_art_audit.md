# 주요 인물 구성·원화 감사

## 1. 감사 기준과 판정 의미

- 서사 정본: `docs/characters/full_character_design_bible.md`, `docs/narrative/prologue_production_script.md`, `docs/narrative/chapter_01_production_script.md`, `data/characters/main_cast.json`, `data/narrative/*.json`.
- 화면 정본: 카툰형 02 — 거친 선화, 명확한 외곽선, 2.5단계 셀 명암, 제한된 회화 질감, 작은 화면에서 읽히는 실루엣.
- `승인`: 해당 파일을 인물 외형 정본으로 후속 제작에 사용 가능.
- `조건부`: 인물·소품 방향은 채택하되 명시 교정 뒤 제작 가능.
- `재작업`: 기획 참고만 가능하며 정본 시트를 새로 그려야 함.
- `거절`: 현재 기획과 충돌하여 정본·파생 제작에 사용하지 않음.

이번 판정은 콘셉트 방향 승인이다. 기존 이미지는 배경이 합쳐진 단일 일러스트이며 턴어라운드, 투명 배경, 표정 분리, 레이어 분리가 없으므로 그대로 2D 게임 애셋으로 승인하지 않는다.

## 2. 인물 구성 정합성

### 즉시 수정이 필요한 문제

1. **주몽 나이 충돌(P0)**: `data/characters/jumong.json`의 `age_at_escape`만 19세다. 제1장 JSON, 전체 바이블, 제작 대본과 `main_cast.json`은 모두 17세다. 제1장 정본은 **17세**이며 향후 데이터 담당자가 19를 17로 통일해야 한다.
2. **다른 왕자들의 실체 부재(P0)**: `other_princes`가 제1장 필수 인물이지만 개별 ID·외형·대사 주체가 없다. 대소 외 왕자 최소 2인을 `무력형`과 `정보형`으로 구분하되 확정 사료처럼 이름 붙이지 말고, 제작용 임시 ID(`prince_martial`, `prince_court`)를 먼저 정해야 한다.
3. **유화의 두 시기 외형 미분리(P0)**: 프롤로그 20대 초중반과 제1장 30대 후반~40세 전후를 한 파일이 대표한다. 머리, 옷의 마모, 청동 장식은 이어지되 얼굴 나이와 자세가 구별된 두 정본이 필요하다.
4. **금와의 두 시기 외형 미분리(P1)**: 프롤로그 40대 후반~50대와 제1장 60대 중후반이 한 파일로 혼용된다. 제1장에서는 희끗한 머리, 깊어진 눈가, 약간 굽은 등으로 시간 경과를 보여야 한다.
5. **제1장 기능 인물 누락(P1)**: 사냥터의 가담 젊은 병사, 얼굴 가린 궁수·창병, 왕성 호위·추격병의 원형이 없다. 젊은 병사는 비살상 선택의 감정적 핵심이므로 일반 병사 복제본으로 처리하면 안 된다.

### 역할 중복과 관계 점검

- 대소와 해명은 겉보기 역할이 겹치지만 유지할 가치가 있다. 대소는 후계 질서를 지키려는 정치적 결정자, 해명은 명령을 살해 지시로 확대하는 실행자다. 해명의 충성 대상을 `대소 개인`이 아니라 `동부여의 안정`으로 유지해야 평면 악역 중복을 피한다.
- 왕성의 신하는 보수적 위험 경고자이고 협보는 기록 조작을 거부하는 실무자다. 둘 다 기록 소품을 들지만 세대·권한·선택이 다르므로 역할은 중복되지 않는다. 실루엣은 신하의 넓은 소매/목간 묶음과 협보의 짧은 작업복/약초·연막 도구로 분리한다.
- 어부와 나루 책임자는 구조자/보고 책임자로 기능이 명확히 나뉜다. 나루 책임자의 복색을 왕실 남색으로 만들면 신하와 혼동되므로 짙은 갈색을 유지한다.
- 오이·마리·협보의 제1장 등장은 `간접 도움`에 한정한다. 이 시점부터 주몽의 충성스러운 파티처럼 연출하면 관계 성장의 설득력이 사라진다.
- 유화의 동기는 일관된다. 정치적 진실을 밝히는 것보다 주몽의 생존과 자기 이름의 선택권을 우선하며, 제1장 장면 7에서 능동적으로 탈출을 명령한다. 수동적으로 우는 어머니 포즈는 금지한다.
- 금와는 보호자이면서 통치자다. 암살 공모자나 무능한 노왕으로 보이지 않게 `직접 심문` 명령과 왜곡된 전달을 시각적으로 구분해야 한다.

## 4. 오프닝·제1장 추가 일러스트 우선순위

### P0 — 실제 오프닝/제1장 제작 전 필수

1. **주몽 17세 턴어라운드**: 정면·측면·후면, 합성궁/활집/팔보호대 분리, 낮은 묶음머리, 손-활 크기표.
2. **유화 2시기 비교 턴어라운드**: 프롤로그 젖은 상태와 제1장 실내복. 같은 청동 장식의 형태를 고정한다.
3. **금와 2시기 비교 시트**: 중년 구조 보고 장면과 노년 궁술 시험·심문 장면. 머리 장식과 지휘봉 규격 고정.
4. **대소 턴어라운드/표정 시트**: 공개석상 절제 미소, 칭찬을 들은 불안, 사적 냉정, 명령 후 흔들림. 왼 눈썹 흉터 고정.
5. **제1장 다른 왕자 2인 시트**: 무력형/정보형의 얼굴·키·어깨·무기 분리. 대소와 동일 얼굴 복제를 금지한다.
6. **유화 탈출 명령 컷신 키프레임**: 청동 장식을 주몽 손에 쥐여 주는 손 클로즈업과 서로 놓지 못하는 시선. 수동적 눈물 연출 금지.
7. **주몽 궁술/사냥터 포즈 시트**: 바람 확인, 만작, 정밀 사격, 밧줄 절단, 비살상 제압, 부상 병사에게 활을 내린 자세.
8. **기능 조연 턴어라운드**: 어부, 나루 책임자 재설계, 신하, 무골, 아란 재설계, 해명. 각자의 직업 소품을 별도 레이어로 제공.

### P1 — 대화와 컷신 완성도에 필수

1. 대화 초상화: `dialogue_portraits.json`의 각 표정을 흉상 3/4 구도로 제작. 우선순위는 주몽 → 유화 → 대소 → 금와 → 해명 → 무골/아란 → 오이/마리/협보 → 프롤로그 3인.
2. 오프닝 컷신: 강물 속 유화 실루엣, 어부의 호흡 확인, 나루 보고, 금와의 판단, 청동 장식 클로즈업, 17년 전환 화살.
3. 제1장 컷신: 금와의 공개 칭찬/대소 반응 투샷, 닫힌 왕자 회의, 유화 방의 모자 대립, 대소-해명 명령 장면, 조작된 목간, 닫히는 성문과 유화의 창.
4. 가담 젊은 병사 고유 흉상과 가면 전/후. 일반 적과 얼굴·체형을 구분한다.
5. 오이·마리·협보 간접 도움 포즈: 시비로 시선 끌기, 수레로 길 막기, 조용히 기록 오류 경고. 아직 파티 단체 영웅 포즈를 사용하지 않는다.

### P2 — 이후 확장 준비

- 오이·마리·협보 턴어라운드와 전투 포즈 풀세트.
- 동부여 왕실 호위, 추격병, 왕성 생활인, 얼굴 가린 매복병의 모듈러 원형.
- 소서노·연타발·바리·비류찬은 제2장 이후 제작 승인 게이트에서 재검토한다.

## 5. 제작팀 전달용 불변 규칙

- 제1장 주몽은 17세이며 왕관·태양·삼족오 문장을 쓰지 않는다.
- 카툰형 02의 외곽선과 제한 명암을 먼저 맞추고 회화 질감은 재질 강조부에만 쓴다.
- 세력색은 동부여 남색/청흑, 주몽 적갈/적동, 졸본 흙빛/녹갈로 유지한다.
- 동일 청동 장식, 주몽 활, 대소 창, 해명 창과 뿔피리는 인물별 고정 모델 시트를 먼저 승인받는다.
- 유화는 행동하는 생존자, 금와는 정치적 한계가 있는 보호자, 대소는 유능한 후계 불안의 소유자로 표현한다.
- `cartoon_v1`과 `main_cast_v1`의 얼굴을 섞어 파생하지 않는다. 표의 정본 후보를 기준으로 턴어라운드를 새로 확정한 뒤 모든 초상화·스프라이트를 파생한다.

## 6. 최종 판정

- 서사 구성: **조건부 승인**. 중심 동기와 관계는 명확하나 주몽 나이 충돌, 다른 왕자 개체화, 두 시기 외형 분리가 선행되어야 한다.
- 오프닝 6인 콘셉트: `cartoon_v1` 5인 조건부, 나루 책임자 재작업.
- `main_cast_v1`: 캐릭터 내용 참고로는 유효하나 카툰형 02 최종 원화로는 전원 조건부 또는 재작업. 그대로 게임 애셋화 금지.
- 다음 승인 게이트: P0 턴어라운드와 유화 탈출 명령 키프레임을 동일 축척으로 나란히 검토한다.
