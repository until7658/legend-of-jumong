# 오프닝·제1장 맵 프리뷰 설계

## 오프닝: 새벽의 강 하류 나루

- 서쪽 절반은 빠른 물살과 얕은 물가로 구성한다.
- 중앙 물가의 돌 여울은 유화가 구조되는 지점과 어부의 접근 동선을 암시한다.
- 동쪽 육지는 나루 책임자의 공간으로 이어질 수 있는 열린 길을 남긴다.
- 수목은 강변 시야를 막지 않도록 외곽에 배치하고, 바위는 구조 지점의 깊이 기준으로 사용한다.
- 새벽의 청회색 조명을 사용하며 초자연광은 표현하지 않는다.

## 제1장: 동부여 왕성 외곽 훈련장

- 남북 길은 왕성 내부와 궁술장을 연결하는 공개 동선이다.
- 동쪽의 넓은 흙 공터는 궁술 시험과 군사들의 관람 공간이다.
- 북쪽 절벽선은 사냥터와 매복 능선으로 이어지는 위험 구역을 예고한다.
- 나무와 바위는 궁술 시야를 방해하지 않도록 외곽에 배치한다.
- 향후 표적, 군사, 주몽과 대소를 독립 오브젝트로 추가할 수 있도록 중앙을 비워 둔다.

## 현재 제한

- 두 씬은 공간·분위기 확인용 독립 프리뷰이며 메인 시작 흐름에는 연결하지 않는다.
- 대사, 컷신, 플레이어 이동, 충돌과 전투는 포함하지 않는다.
- 생성형 타일 경계는 최종 Terrain Set 후처리 전이므로 일부 이음선이 보일 수 있다.
