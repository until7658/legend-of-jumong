# AI 생성 애셋 기록

> 전체 2D 전환 상태: 과거 항목의 `HD-2D` 표현은 생성 당시 목적을 보존한 이력이다. 해당 애셋은 Node3D·Sprite3D·3D 재질로 통합하지 않으며, 실제 채택 시 Sprite2D·TileMapLayer·CanvasItem 기반 전체 2D 규격으로 재검토한다.

- 파일: `assets/tiles/golden/cartoon_northern_ground_gold_v1.png`, `assets/objects/golden/cartoon_northern_props_gold_v1.png`

- 분류: `stylized-concept`
- 목적: `cartoon_v1` 캐릭터 원화와 맵의 선·색면·명암·광원 언어를 맞추기 위한 환경 골든 스타일 승인 후보
- 스타일 참고: `assets/concept/cartoon_v1/jumong.png`, `yuhwa.png`, `king_geumwa.png`
- 참고 범위: 가늘고 날카로운 선, 각진 큰 색면, 2.5단 셀 채색, 제한적인 마른 붓 질감, 청흑 암부와 얇은 호박 림라이트만 참고했다. 인물·얼굴·복식·장비·포즈·문양은 사용하지 않았다.
- 지면 구성: 저채도 풀, 다져진 흙, 얕은 회청색 물, 층상 바위와 접경을 한 화면에 구성

- 규격: 지면 1254×1254 RGB PNG, 자연물 1536×1024 RGB PNG
- 주요 제한: 인물·건물·문자·워터마크·사진적 미세 노이즈·3D 렌더·특정 게임 및 작가 모사 배제
- 권리 검토: 프로젝트 내부 생성 원화를 스타일 참고로 사용했으며 고유 인물 요소를 환경에 포함하지 않았다. 상업 채택 전 생성 도구 이용 조건과 외부 작품 유사성을 재검토해야 한다.
- 사용 위치: 스타일 검토 문서와 향후 통합 골든 샘플의 참고 입력. 게임 미통합
- 상태: 파일명은 승인 요청 규격을 따르지만 최종 Terrain Set이 아닌 **사용자·PM 승인 후보**. 자연물은 불투명 배경이며 실제 게임용 투명 오브젝트로 직접 사용하지 않는다.
- 검토: `docs/art/map_golden_sample_review.md`
