## 공통 제작 정보

- 제작 도구: Codex의 텍스트 편집을 통한 직접 SVG 작성
- 제작일: 2026-07-17
- 생성형 이미지 도구: 사용하지 않음
- 수정 여부: 신규 제작, 외부 원본 및 기존 테스트 애셋 미사용
- 권리 검토: 기본 도형과 직접 선택한 색상으로 구성한 프로젝트 전용 원본 플레이스홀더다. 특정 게임·드라마·영화의 캐릭터, 문양, 실루엣 또는 화면 구성을 복제하지 않았다. 상업 배포용 최종 애셋은 별도 권리 검토와 교체가 필요하다.

## 표현 원칙

- 캐릭터는 청색 외투와 주황색 중심면만 사용한 무명 인물로, 주몽의 최종 디자인이 아니다.
- 나무와 지면은 원근, 정렬, 가림, 광원 가독성을 시험하기 위한 단순 색면이다.
- SVG는 Godot가 바로 텍스처로 가져올 수 있고 원본 해상도 변경도 쉽도록 선택했다.
- 최종 제작 단계에서는 캐릭터 방향별 프레임, 픽셀 밀도, 피벗, 색 공간, 라이트 반응 방식을 별도 사양으로 확정한다.

## 통합 가이드

1. `hd2d_ground.svg`를 3D 평면의 `StandardMaterial3D.albedo_texture`에 연결한다.
2. `hd2d_hero.svg`와 `hd2d_tree.svg`는 `Sprite3D`에 연결하고 `billboard`를 활성화한다.
3. 캐릭터의 발 피벗은 텍스처 하단 중앙으로 맞추고 지면과 겹치지 않도록 월드 Y 위치를 조정한다.
4. 마커는 반투명 정렬 문제를 피하도록 캐릭터보다 낮은 렌더 우선순위에서 먼저 시험한다.
