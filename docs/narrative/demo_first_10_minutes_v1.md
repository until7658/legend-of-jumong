# M4 데모 첫 10분 흐름 v1

## 목표

10분 안에 `신화적 탄생`, `주몽 개인의 궁술`, `지휘관+다수 병사 분대전`이라는 세 가지 약속을 전달한다.

| 시간 | 내용 | 연출 레이어 | 완료 신호 |
|---|---|---|---|
| 00:00~00:40 | 타이틀, 언어·화면·게임패드 확인 | UI | 새 게임 선택 |
| 00:40~02:10 | 유화 구조와 주몽 탄생을 압축한 14컷 프롤로그 | 서사 컷신 | 프롤로그 종료/건너뛰기 |
| 02:10~04:20 | 동부여 수련장, 이동·상호작용, 주몽의 궁술 재능 소개 | 개인·필드 컷신 | 훈련대장 대화 |
| 04:20~05:10 | 주몽이 수련 궁병 8명을 배속받고 거리 상성을 배움 | 전술 튜토리얼 | 첫 분대 선택 |
| 05:10~08:40 | 주몽 궁병분대 대 훈련대장 창병분대, 이동·사격·방어·턴 종료 | 전술 맵+짧은 교전 컷인 | 상대 지휘관 제압 |
| 08:40~10:00 | 결과, 병사 생존·기세 요약, 체크포인트 저장, 다음 장 훅 | 결과 UI·서사 | `m4_training_complete` 저장 |

## 연출 원칙

- 전술 맵은 지휘관과 병사 8명을 한 조작 단위로 다룬다.
- 공격 순간의 다수 병사 충돌은 별도 교전 컷인으로 확장한다. 반복 시 건너뛰기·고속 옵션을 제공한다.
- 궁술대회는 분대전과 분리해 주몽 개인의 자세·호흡·화살 궤적과 관중 반응을 보여 준다.
- 훈련전의 패배·손실은 사망이 아니라 제압·후퇴로 기록한다.

## 현재 프로토타입 상태

- 구현: 타이틀, 압축 프롤로그, 지휘관+병사 8명 쿼터뷰 격자 전투, 거리 상성, 세이브 체크포인트
- 블록아웃: 분대 캐릭터와 지면, 전투 HUD
- 다음 구현: 수련장 자유 이동, 훈련대장 대화, 공격 시 16대16 교전 컷인, 결과 상세 화면
