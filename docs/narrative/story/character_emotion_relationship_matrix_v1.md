# 주요 인물 감정선·관계 변화표

| 인물 | 시작 | 전환 | 도하 | 졸본 | 건국 |
|---|---|---|---|---|---|
| 주몽 | 인정 욕구를 숨기는 17세 이방인 | 타인을 돕느라 도주가 늦어지고 첫 동료를 얻음 | 혼자 책임지려는 결함을 인정 | 혈통 대신 약속 이행으로 신뢰 획득 | 권한을 나누는 지도자 |
| 유화 | 침묵으로 아들을 보호 | 제1장에서 떠나보내며 보호 방식을 바꿈 | 청동 장식과 전갈로 부재 속 기준점 | 직접 등장보다 소식/기억 | 아들의 국가가 자신을 신격화하는 일을 경계하는 여운 |
| 대소 | 유능하지만 후계 불안이 큰 왕자 | 암살 실패를 반역 서사로 바꿈 | 해명의 과잉을 묵인하며 되돌아갈 선을 넘음 | 교역 압박으로 정치적 고립 시도 | 패배보다 `안정` 명분의 균열을 남김 |
| 금와 | 보호와 통치 사이 | 왜곡된 명령을 뒤늦게 인지 | 주몽 추격을 완전히 멈추지 못함 | 직접 개입의 한계 | 전갈/침묵으로 복합적 부성 유지 |
| 오이 | 왕성 사람 불신 | 주몽의 주민 보호를 보고 임시 동행 | 방패로 마지막 도하를 지킴 | 공동체 방어 책임 수용 | 군사 권한을 공동 규칙 아래 맡음 |
| 마리 | 손익을 계산하는 길잡이 | 압수 물자보다 사람을 먼저 고른 주몽 평가 | 물길·교역 지식으로 도하 설계 | 졸본 교역로 연결 | 이동과 교역의 책임자 |
| 협보 | 결정을 늦추는 기록 실무자 | 조작 장부 공개로 돌아갈 길을 끊음 | 도하 순서와 물자 기록 | 집단 간 약속을 문서화 | 새 나라의 기록이 신화를 왜곡할 위험을 제기 |
| 소서노 | 외부인을 비용과 위험으로 평가 | 해당 없음 | 해당 없음 | 약속을 지키는지 시험하고 조건부 동맹 | 운명의 배우자가 아니라 공동 설계자 |
| 연타발 | 환대와 경계 | 해당 없음 | 해당 없음 | 물자·노동·책임 시험 | 권력보다 지속 가능한 분배를 요구 |
| 비류찬 | 외부인 반대 | 해당 없음 | 해당 없음 | 주몽과 방어권 충돌 | 지휘권을 공유하며 협력 |
| 바리 | 영웅담 불신 | 해당 없음 | 주민 우선 약속을 행동으로 확인 | 강가 공동체 연결 | 도하 신화가 과장되는 것을 웃으며 경계 |
| 해명 | 질서 명분의 실행자 | 병사를 소모하며 대소와 차별화 | 무리한 추격으로 명분 훼손 | 대소 명령의 칼날 | 개인 결말은 별도 승인 필요 |

## 핵심 관계 단계

- 주몽↔유화: 보호받는 아들 → 반발 → 생존을 명령받은 계승자 → 부재 속 가치의 계승.
- 주몽↔대소: 형제 호칭의 긴장 → 정치적 경쟁 → 조작된 반역자/추격자 → 서로 다른 국가 질서의 대립.
- 주몽↔오이·마리·협보: 간접 도움 → 거래적 협력 → 상호 검증 → 동료 → 공동 창건자.
- 주몽↔소서노: 경계받는 난민 → 조건부 협력자 → 약속을 증명한 동맹 → 권한을 나누는 공동 설계자.
